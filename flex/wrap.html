# 换行

# flex-wrap

flex-wrap 属性的作用是限制弹性容器只显示一行，或是在必要时显示多行。

  • 默认值为 nowrap，不换行
  • 允许换行时
    • wrap 第二行在第一行之后（块级元素的排列方向）
    • wrap-reverse 第二行在第一行之前
