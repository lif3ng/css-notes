# flex-direction

`flex-direction`属性通过设置弹性容器的主轴方向，指定弹性子元素在弹性容器中的排列方式。

取值有

- `row` 与书写方向一致（由`writing-mode`, `direction`,`text-orientation`决定）
- `row-reverse` 与书写方向相反
- `column`
- `column-reverse`
