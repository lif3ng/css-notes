# 单位

## 长度

### 绝对单位

---

<Demo name="absolute" />

#### px 像素

像素。对于屏幕显示，1 像素通常表示一个设备像素（点）。然而，对于打印机和高分辨率屏幕，一个 CSS 像素意味着多个设备像素。1px=1 英寸的 1/96。

#### in

英寸。1in = 2.54cm = 96px

#### cm

厘米。在 72ppi 屏幕上用尺子测量无误，在视网膜屏上测量小于 1 厘米。
